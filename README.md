# FinSight AI

FinSight AI is a modern AI-powered financial intelligence and expense analytics platform designed to help users understand, optimize, and manage their financial behavior through real-time analytics, smart insights, and cloud-based financial tracking.

Built using React, Firebase, Firestore, and Vercel, the platform combines responsive fintech dashboard design with intelligent spending analysis to create a professional full-stack personal finance experience.

# Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Saahiti-Rotte/finsight-ai.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

# Environment Variables

Create a `.env` file in the root directory and configure:

```env
VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=
VITE_FIREBASE_MESSAGING_SENDER_ID=
VITE_FIREBASE_APP_ID=
```
